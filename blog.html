Ans to the question 1

Use var when you want a variable to be function-scoped or when working with older code.
Prefer let for variables that need to be block-scoped and may be reassigned.
Use const for variables that need to be block-scoped and should not be reassigned after initialization.

Ans to the question 3

REST API which stands for Representational State Transfer Application Programming Interface, is an architectural style for designing networked applications. It is a set of principles that define how web standards, such as HTTP (Hypertext Transfer Protocol), should be used to create web services. RESTful APIs are designed to be simple, scalable, and stateless, making them well-suited for use in distributed systems and the web
